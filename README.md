# YOUTUBE Video-Downloader

Simple yt-dlp script to download yt videos.
Stop using shitty video recorders to screen record a yt-video.

**Functionalities**
- Name the output video file 
- The progrma automatically installs the best quality form of the yt video


# Installation

- Clone the repository:
```
git clone https://github.com/lxsh-S/yt-downloader.git
```

- Install the dependencies:
```
pip install -r requirements.txt
```

- Run 'main.py':
```
python main.py
```
